# Dok-Fu Format Reference

Canonical format specification for doc modules, code pointers, and the index.

## Module boundaries

One doc module per **source directory**. A module documents a folder, not a single file. Each H2 section inside the module documents exactly one source file within that folder. Root-level files are documented in `docs/root.md` with `code: .` in the frontmatter.

## Doc module frontmatter

```yaml
---
dokfu_id: <stable-slug>
code: <repo-relative-path-to-source-folder>
tags: [<tag1>, <tag2>]
description: <one sentence>
---
```

- `dokfu_id`: slugified folder path (e.g. `src-auth`); used for rename repair
- `code`: repo-relative path to the **source folder** this module covers; must resolve to an existing directory
- `tags`: values must exist in `config/tags.registry.json`
- `description`: one sentence; this is what appears in the index

## Doc module body

```
# <Component/Folder Name>

## Sections
- [filename.py](src/component/filename.py)
- [other.py](src/component/other.py)

## filename.py
path: src/component/filename.py
<content — max 3 sentences and 5 bullets>

## other.py
path: src/component/other.py
<content — max 3 sentences and 5 bullets>
```

The `## Sections` block lists the module's H2 headings — one per source file in the folder. Each H2 section header is the **filename**, and the first line of the section body is `path: <repo-relative path>` to that specific file.

Section links use relative paths to the source file (e.g. `[login.py](src/auth/login.py)`).

## Code pointer

A single comment line placed near the top of **each source file** (after shebang/package, before imports). Points to the parent module (folder-level doc):

```
# dok-fu: docs/<mirrored-folder-path>.md
```

The canonical format shows `# dok-fu:` with a space after `#`. The comment token is determined by file extension using `comment_map` in `config/dok-fu.config.json`. All files within the same folder point to the same module doc.

## Index entry

Flat JSON array in `docs/index.json`. Each entry:

```json
{ "path": "docs/...", "tags": ["tag1"], "description": "One sentence." }
```

Never hand-edited. Regenerated by `dokfu index`.
