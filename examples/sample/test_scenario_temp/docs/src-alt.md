## greeter.py
path: nonexistent.py
Provides `greet()` and `farewell()` functions that format human-readable greeting strings.
- `greet(name)` → `"Hello, <name>!"`
- `farewell(name)` → `"Goodbye, <name>!"`

## calculator.py
path: src/calculator.py
Implements basic arithmetic: addition, subtraction, and multiplication over floats.
- `add(a, b)` → sum
- `subtract(a, b)` → difference
- `multiply(a, b)` → product
